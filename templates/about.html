About Articulax

Articulax is a speaking practice platform designed to help users improve communication skills through structured practice sessions and feedback.

How it works

1. Choose a mode

Users will eventually be able to practice Communication, Interview, and Presentation modes.

2. Start a session

Pick a topic, set your audience and tone, and begin speaking with microphone access enabled.

3. Review feedback

After a session, users can view scores, session history, and feedback to improve over time.
